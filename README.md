# 辛いもののランキングをみんなで作るアプリ

## 概要
みんなから辛いものの評価（これはあれより辛かったけど、あれよりは辛くなかった）を集め、その評価をもとに全体のランキングを計算・表示するアプリケーション

## 使用技術
- Flutter 3.10.0
- Dart 3.0.0
- DevTools 2.23.1
